# population-trend-app

REASAS-API を利用した人口推移グラフの表示アプリです。
https://population-trend-app.vercel.app/

## 環境

- Next.js
- Typescript
- recoil
- swr
- axios
- Emotion
- chart.js
- react-chartjs-2

## セットアップ

```
yarn install
yarn dev
```
